1.- Solicitamos una url y enviaremos datos mediante GET o POST


Si la peticion es GET: /*
	Buscamos Archivo Estatico

Si la peticion es GET: /{Controlador}/{Vista}/{[args]}
	Opcion 1: Buscamos Controlador / Vista
	Opcion 2: Asignamos Controlador Default y respondemos como ORM
	

si la peticion es GET: /{modelo}/API/
	Respondemos con modelo en formato json.
